  1. b : this tag is use for bold text
  2. i : this tag is use for italic text
  3. u : this tag is use for underline text

    Example : a2 + b2 + 2ab
  5. sub : this tag is use to write subscript
    Example : H2O
  6. em : this tag is use to write emphasis text
    Example : hello
  7. lorem : this tag writes some words automatically according to the number of words we want to write
    Example : lorem20
  8. del : this tag is ue to write text which is deleted from the document
    Example : hello
  9. abbr : this tag is use to write abbreviation of a word
    Example : HTML
